# Pasale Dai
*Your Friend in Need*

## Prerequisites
1. Node.js
	* Windows: [Node.js](https://nodejs.org/en/)
	* Linux:
		```
		sudo apt-get install node
		sudo apt-get install npm
		```
	* Mac:
		```
		brew install node
		```

## Project Setup
1. Open terminal at file location and type `node app`
2. Open browser and type `localhost:3000`
